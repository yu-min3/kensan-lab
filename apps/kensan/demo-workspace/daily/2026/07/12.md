# 2026-07-12

## 日記

週末を使って Longhorn の replica 配置を見直した。Pi の microSD に replica が乗ると I/O で全体が詰まるのを実測で確認できたのが収穫。NVMe ノード限定に寄せてから体感が明らかに軽い。

## 学習

- Longhorn の `dataLocality: best-effort` は「読み手と同じノードに replica を寄せる」だけで、強制ではない。強制したいなら `strict-local` だが単一ノード障害に弱くなる
- RecurringJob の group 指定で全 volume を一括バックアップ対象にできる。app 側の per-PVC 設定が不要になるのが良い
