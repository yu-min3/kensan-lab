# 2026-07-13

## 日記

朝のうちに ArgoCD の drift を眺めるのが習慣になってきた。今日は OutOfSync が 0 で気持ちがいい。GitOps に寄せてから「昨日の自分が何をしたか」を Git log で追えるのが効いている。

## 学習

- ArgoCD の server-side diff を有効にすると、mutating webhook が書き込むフィールドの phantom OutOfSync が消える
- `ignoreDifferences` は最終手段。まず「なぜ差分が出るか」を潰す
